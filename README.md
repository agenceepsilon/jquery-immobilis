# jQuery Immobilis

Easily place your content blocks on top or bottom of the page to scroll.

## Parameters

| Options       | Types   | Defaults  | Others     |
| ------------- | ------- | --------- | ---------- |
| mainClass     | string  | immobilis | Your class |
| css           | boolean | ``true``  | ``false``  |
| target        | string  | top       | bottom     |

## Initialize

    $(".mobilis").immobilis();
